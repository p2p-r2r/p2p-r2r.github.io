## Getting Started

1. Clone this repository
```bash
git clone https://github.com/yourusername/p2p-r2r.github.io.git
cd p2p-r2r.github.io
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```
